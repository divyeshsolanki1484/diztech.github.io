## Description

## Get started with the Abandoned Cart plugin:

- Download the Zeo Abandoned Cart plugin from the Community Store
- Activate the plugin in your Shopware 6 installation – this can be done in the Plugin Manager under “Purchases”. Alternatively, you can use the Plugin Manager to upload the plugin ZIP file.
- Finally open the plugin configuration via My Plugins and the three dots after the "Abandoned Cart" plugin.
- Be sure to have the "active" and the toggles for each page setting enabled
