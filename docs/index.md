# Abandoned Cart user guide

## Plugin Information

- Our plugin will send reminders to customers who have abandoned their carts, encouraging them to complete their purchases. This can result in higher sales and revenue for your business.

## Installation of the Plugin

- Go to Extension
- Install the Plugin
- The Plugin will be installed and activated.

## Plugin Configuration

Navigate to Extensions -> My Extensions -> Plugin configuration there will many options:

**Active Switch**
Enable the active switch to send reminder emails to customers who have abandoned their carts.

**Cart recovery is anonymous**
We added an anonymous cart option. You can now configure it to enable the abandoned cart email to recover customer data for checkout or simply restore the cart contents.

**Time until the first reminder gets triggered in HOURS. (Fallback in case of an empty schedule)**
- The mail-send delay is only used when no notification schedule is configured.
- This configuration option is deprecated and will be removed in the future.

**Abandoned Cart decay time (Hours)**
Configure a decay time to automatically clear up hanging abandoned carts.

**Max number of notifications to be sent out per processing cycle:**
You can set the maximum number of emails to be sent out per processing run to prevent your mail server from marking them as spam.

### Configure the time schedule

- Time schedule instructions
    1. Add a new row using the add row button
    2. Enter a value (in minutes) in the delay column.
    3. Pick a mail template to be used, be sure this mail template contains valid HTML and no illegal variables.
    4. Repeat steps 1 to 4.
- You can delete one or multiple rows by selecting the checkbox. A "Delete" button will appear which allows you to delete the selection. Your changes will be permanent once you hit the "Save" button in the top right.

## Cart conversions

- Go to Marketing > Abandoned Carts
- By this admin can gain insight into cart conversions within your store and keep track of reminders being triggered.
- After placing an order, It will remove the order from the abandoned cart list.

## Abandoned Cart conversions on the Dashboard

- Go to Dashboard > Scroll down to the Abandoned Cart Conversion
- The orders that are placed from the abandoned cart list will be displayed here.
- Admin can also know the recovered orders from abandoned carts and the conversion rate of orders till now.
