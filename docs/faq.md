# Frequently Asked Questions

## How to configure the time schedule settings?

Go to Extensions -> My Extensions -> Plugin configuration -> Scroll down to  time schedule settings

1. Add a new row using the add row button
2. Enter a value (in minutes) in the delay column.
3. Pick a mail template to be used, be sure this mail template contains valid HTML and no illegal variables.
4. You can delete one or multiple rows by selecting the checkbox. A "Delete" button will appear which allows you to delete the selection. Your changes will be permanent once you hit the "Save" button in the top right.

## What is abandoned cart conversion?

By this admin can gain insight into cart conversions within your store and keep track of reminders being triggered. After placing an order, It will remove the order from the abandoned cart list.
(Admin -> Marketing -> Abandoned carts)

## Can I view the abandoned cart conversion on the dashboard?

Yes, Go to Dashboard -> Scroll down to the Abandoned Cart Conversion
The orders that are placed from the abandoned cart list will be displayed here.
Admin can also know the recovered orders from abandoned carts and the conversion rate of orders till now.
